Была разработана 3D модель автомобиля в программе Ansys electronic. Кузов автомобиля выполнен из алюминия. 

 Unity пружинный маятник
 
 В ходе выполнения работы в Unity была разработана симуляция пружины, демонстрирующая закон Гука и гармонические колебания. На сцене созданы три основных объекта: точка подвеса Anchor, груз Weight с компонентом Rigidbody и визуальная пружина Spring. Для соединения груза с точкой подвеса использован компонент Spring Joint, настроенный с параметрами жесткости и демпфирования. Движение груза ограничено только вертикальной осью для корректной работы механизма. Разработан скрипт SpringController на C#, который обрабатывает ввод с клавиатуры, позволяя нажатием пробела создавать импульс для раскачивания пружины, а стрелками изменять жесткость в реальном времени. Скрипт также динамически обновляет визуальное отображение пружины, изменяя ее масштаб и положение в зависимости от расстояния до груза. Для наглядности создан пользовательский интерфейс со слайдером регулировки жесткости и текстовыми полями, отображающими отклонение, скорость и текущие параметры. В ходе настройки были решены технические проблемы: конфликт систем ввода решен переключением на режим Both, а для плавности движения увеличены настройки точности физики и включена интерполяция Rigidbody. В результате симуляция наглядно показывает, как груз совершает затухающие колебания при импульсе, а изменение жесткости влияет на частоту колебаний. Визуальная пружина реалистично растягивается и сжимается, а информационная панель позволяет отслеживать числовые характеристики движения. Проект полностью демонстрирует физическое явление и может использоваться в образовательных целях.
